📚 Panduan & Penjelasan Lengkap Airdrop

Pahami dasar-dasar, alat tempur, hingga istilah penting dalam dunia airdrop crypto langsung dari sudut pandang ahlinya.

Penjelasan Apa Itu Airdrop?

Menurut saya pribadi, airdrop adalah sebuah acara event berhadiah token gratis dengan mengerjakan tugas social media. Developer meminta kita untuk memfollow semua social media nya untuk mendapatkan token tersebut. Selain bertujuan untuk mendapatkan token, developer membuat event tersebut untuk menarik investor dan mencari komunitas supaya project tersebut bisa terkenal di kalangan banyak orang.

Durasi Acara?

Biasanya 1 - 3 month, ada juga yang sampai 1 - 3 tahun tergantung dev nya.

Kapan Airdrop Distribusi?

Paling cepat hitungan hari , hitungan minggu - 3 bulan, Paling lama 1 - 3 tahun

Pakai Wallet Apa?

1. Metamask Wallet tersebut multi fungsi dan banyak network, misalkan dev meminta address ERC20 kalian cukup COPY Address ETH, begitu pula address BEP20 / BSC , Cukup Copy Address ETH Trustwallet, karena sama cuma beda Network.
2. Tronlink Wallet tersebut digunakan di jaringan TRC20, misalkan developer meminta input address TRX TRC20 , cukup COPY address TRX.
3. TokenPocket Wallet tersebut digunakan di jaringan HECO , HECO merupakan jaringan Huobi Eco Chain, misalkan developer meminta input address HT / HRC20 , kalian cukup Copy Wallet HT , wallet HT mirip dengan ERC20 & BEP20 yang membedakan cuma network jaringan.
4. Wombat Wallet / Eoslynx Wallet tersebut digunakan di jaringan EOS, jarang sekali airdrop di Platform EOS, misalkan ada airdrop dengan wallet EOS , kalian cukup copy username walletnya, disini untuk mendapatkan full akses kalian harus membayar beberapa dollar.
5. Bitget Wallet Sama dengan metamask tapi jumlah wallet sedikit.
6. SafePal Wallet multi fungsi seperti bitget.

Bagaimana Menjual Token Tersebut?

Kalian cukup paste address smart contract / sc kemudian pasang Slippage 1% - 2.5% apabila Liquidity puluhan ribu sampai ratusan ribu dollar. Slippage 5% digunakan apabila Liquidity kecil atau gede dan kalian menginginkan transaksi di proses cepat, hal ini ber efek ETH / USDT / BSC yang kalian dapatkan berkurang.

📌 Pengaruh Gas GWEI: Gas GWEI saat swap juga berpengaruh di dalam antrian, misalkan anda swap di jaringan ETH cukup ditambah +5 s/d +10 gwei. Misalkan anda swap di jaringan BSC cukup pakai gas gwei seperlu nya saja.

Glosarium Crypto

Profit : Keuntungan yang di dapat.

Rekt : Kamu rugi tidak dapat untung.

Buy : Membeli sebuah token / coin di Market / Exchange

Sell : Menjual token / coin untuk dijadikan fiat / dana

Hold : Menahan token / coin di wallet atau Exchange dikarenakan menunggu harga tinggi

Pump : Token tersebut naik tinggi keatas

Dump / Koreksi : Token tersebut turun kebawah

Maintenance : Perbaikan yang di lakukan oleh exchange atau developer untuk meningkatkan server

Scam : Project tersebut adalah penipuan yang dilakukan oleh dev.

Rugpull : Developer menjual semua token di market / Exchange dan membuat harga token tersebut dump parah , bisa dibilang scam

Ripper : Penipu, biasanya setelah anda kirim uang , anda akan di block

FUD : Menyebarkan ketakutan dan informasi yang salah untuk mendapatkan keuntungan supaya token tersebut dump, serta memiliki tujuan untuk mempengaruhi sentimen pasar sehingga harga mengalami koreksi. Biasanya terjadi pada saat harga sudah melambung tinggi dan stagnan pada level tertentu, emosi ini menjadi dorongan bagi para pelaku pasar untuk buru-buru menjual suatu koin supaya tidak keburu turun.
