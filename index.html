Penjelasan Apa Itu Airdrop?

Menurut saya pribadi, airdrop adalah sebuah acara event berhadiah token gratis dengan mengerjakan tugas social media. Developer meminta kita untuk memfollow semua social media nya untuk mendapatkan token tersebut. Selain bertujuan untuk mendapatkan token, developer membuat event tersebut untuk menarik investor and mencari komunitas supaya project tersebut bisa terkenal di kalangan banyak orang.

Durasi Acara?

Biasanya 1 - 3 bulan, ada juga yang sampai 1 - 3 tahun tergantung regulasi developer proyek.

Kapan Airdrop Distribusi?

Paling cepat hitungan hari atau minggu - 3 bulan. Paling lama berkisar antara 1 - 3 tahun.

Pakai Dompet (Wallet) Apa?

1. Metamask Wallet tersebut multi fungsi dan mendukung banyak network. Misalkan dev meminta address ERC20 kalian cukup COPY Address ETH, begitu pula address BEP20 / BSC. Cukup Copy Address ETH Trustwallet, karena formatnya sama hanya berbeda integrasi Network.
2. Tronlink Wallet tersebut digunakan di jaringan TRC20, misalkan developer meminta input address TRX TRC20, cukup COPY address TRX utama Anda.
3. TokenPocket Wallet tersebut digunakan di jaringan HECO (Huobi Eco Chain). Misalkan developer meminta input address HT / HRC20, kalian cukup Copy Wallet HT. Wallet HT mirip dengan ERC20 & BEP20 yang membedakan cuma konfigurasi network jaringan.
4. Wombat Wallet / Eoslynx Wallet tersebut digunakan di jaringan EOS. Jarang sekali ada event airdrop di Platform EOS. Misalkan ada, kalian cukup copy username walletnya. Biasanya untuk akses penuh diperlukan aktivasi berbayar beberapa dollar.
5. Bitget Wallet Sama fungsionalitasnya seperti Metamask dengan opsi manajemen multi-chain terintegrasi.
6. SafePal Dompet multi-fungsi dengan keamanan tinggi, setara dengan kapabilitas Bitget maupun Metamask.
